# Changelog

All notable changes to this project are documented in this file.

## 0.1.0 - 2026-07-09

- Added production-ready milk tea storefront flows.
- Added i18n infrastructure for Vietnamese and English.
- Added CMS-ready site configuration with local admin overrides.
- Added dynamic theme engine and reusable design system primitives.
- Added SEO metadata, PWA manifest, service worker registration, sitemap, and robots file.
- Added `/admin` configuration panel for editable storefront content.
- Removed unused runtime dependencies.
- Added project documentation, contributing guidelines, and MIT license.
