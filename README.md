# MilkTea Premium

A premium Bubble Tea landing page built with Vite, Tailwind CSS, and vanilla JavaScript.

## Features

- Modern responsive UI inspired by Apple, Starbucks, and Gong Cha
- Sticky navigation
- Product categories and featured products
- Promotion banner
- Testimonials and Instagram gallery
- Newsletter and SEO-ready meta tags

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```
