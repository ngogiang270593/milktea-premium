# ✨ Overview

**MilkTea Premium** is a modern commerce starter kit built for food & beverage businesses.

Instead of being limited to a single Bubble Tea demo, the project is designed as a reusable foundation that can be quickly customized for different business models, including:

- 🧋 Bubble Tea Shop
- ☕ Coffee Shop
- 🥐 Bakery
- 🍔 Restaurant
- 🍹 Beverage Brand
- 🍰 Dessert Store

Built with **Vite**, **Vanilla JavaScript**, and **Tailwind CSS**, the project focuses on clean architecture, fast performance, and a modern user experience.

# 🏗 Architecture

The project follows a modular architecture designed for scalability and maintainability.

```text
src/
│
├── assets/
├── components/
├── config/
├── constants/
├── layouts/
├── locales/
├── pages/
├── repositories/
├── services/
├── store/
├── utils/
└── main.js
```

### Architecture Highlights

- Component-based UI
- Repository Pattern

- Global Stores
- Reusable Utilities

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/ngogiang270593/milktea-premium.git
```

Enter the project

```bash
cd milktea-premium
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Open

```text
http://localhost:5173
```

---

# 📦 Production Build

Build the project

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

The production files will be generated inside:

```text
dist/
```
