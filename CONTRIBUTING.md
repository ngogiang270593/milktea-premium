# Contributing

Thank you for considering contributing to MilkTea Premium.

## Getting Started

Clone the repository

```bash
git clone https://github.com/ngogiang270593/milktea-premium.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```
